# CS 1.6 - Team Selector
A plugin that allows you to easily select a team and choose whether to open the MOTD window or not using CVAR.

# Installation
- Just download the plugin and upload the .amxx file to your plugins folder on your server (or you can of course compile the .sma file and then upload the compilated .amxx file to your server).
- Then write the plugin name (with .amxx) to `/cstrike/addons/amxmodx/configs/plugins.ini`.

# Requirements
- AMX Mod X 1.10

# CVARs
`ts_show_motd 1/0` = Shows or hides the welcome MOTD window.

# Default Config File:
```
; Here you can rewrite the names of the teams.
;
Terrorists="Terrorists"
Counter-Terrorists="Counter-Terrorists"
Spectators="Spectators"
```
- You only rewrite what is behind the equal sign (=) in double quotation marks.
